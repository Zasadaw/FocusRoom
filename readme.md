FocusRoom — это веб-приложение для совместных фокус-сессий и управления задачами. Пользователи могут создавать комнаты, запускать таймеры концентрации, отслеживать прогресс и работать вместе в реальном времени.

О проекте

FocusRoom помогает пользователям сосредоточиться на учебе, работе или личных задачах.

Приложение объединяет:

личные задачи;
таймер по типу Pomodoro;
совместные фокус-комнаты;
статистику продуктивности;
сохранение прогресса между сессиями.
Главная идея проекта — создать цифровое пространство, где можно работать одному или вместе с другими людьми, не отвлекаясь.

Цель проекта

Создать приложение, в котором пользователи смогут:
выполнять задачи по таймеру;
создавать совместные комнаты для работы;
видеть свой прогресс;
сохранять историю продуктивности;
повышать мотивацию за счет совместной активности.

Кто пользователь?

Пользователь — это человек, который хочет лучше концентрироваться во время учебы или работы.
Примеры пользователей:
студент, готовящийся к экзаменам;
junior-разработчик, работающий над проектом;
фрилансер, которому важно не отвлекаться;
небольшая команда, которая хочет работать синхронно.
Какую проблему он решает?
Пользователь хочет:
меньше отвлекаться;
структурировать рабочее время;
видеть реальный прогресс;
работать вместе с друзьями, коллегами или одногруппниками;
сохранять результаты своей активности.
Обычный список задач не всегда мотивирует, а FocusRoom добавляет таймер, совместную работу и аналитику.
Что его окружает?
Пользователь обычно находится:
дома;
в библиотеке;
в коворкинге;
в университете;
за ноутбуком или смартфоном.
Контекст использования:
подготовка к экзамену;
выполнение домашних заданий;
работа над проектом;
рабочие сессии в течение дня;
вечерняя концентрация после работы.

Основной функционал

1. Авторизация и профиль пользователя
регистрация;
вход в систему;
хранение данных пользователя;
личный профиль;
сохранение статистики.
2. Личные задачи
создание задачи;
редактирование задачи;
изменение статуса задачи;
удаление задачи;
список активных и выполненных задач.
3. Таймер фокус-сессии
запуск таймера работы;
запуск таймера перерыва;
настройка длительности;
поддержка режима Pomodoro;
автоматическое завершение этапа.
4. Фокус-комнаты
создание комнаты;
приглашение участников по ссылке или коду;
отображение участников комнаты;
общий просмотр активности;
совместная работа в реальном времени.
5. История и статистика
сохранение завершенных сессий;
подсчет общего времени концентрации;
история выполненных задач;
статистика по дням и неделям.

Архитектура приложения

Приложение состоит из двух основных частей:
Frontend — отвечает за интерфейс, таймер, задачи, комнаты и отображение статистики;
Backend — отвечает за авторизацию, хранение данных, обработку задач, комнат и истории сессий.
Логика работы:
Пользователь входит в аккаунт.
Создает или открывает задачи.
Запускает фокус-сессию.
Может присоединиться к комнате или создать свою.
По завершении сессии результат сохраняется в базе данных.
Статистика обновляется в профиле.

API Endpoints

Аутентификация
POST /api/auth/register — регистрация
POST /api/auth/login — вход
GET /api/auth/me — текущий пользователь

Задачи
GET /api/tasks — получить задачи
POST /api/tasks — создать задачу
PATCH /api/tasks/{id} — изменить задачу
DELETE /api/tasks/{id} — удалить задачу

Комнаты
GET /api/rooms — список комнат
POST /api/rooms — создать комнату
POST /api/rooms/join — войти в комнату
GET /api/rooms/{id} — получить данные комнаты

Сессии
POST /api/sessions/start — начать сессию
POST /api/sessions/finish — завершить сессию
GET /api/sessions/history — история сессий

Профиль
GET /api/profile — профиль пользователя
GET /api/profile/stats — статистика
GET /api/profile/recent — последние сессии
WebSocket события
Клиент отправляет
room:join
room:leave
session:start
session:pause
session:end
status:update
Сервер отправляет
room:members_updated
room:session_updated
room:user_status_changed

Заключение

FocusRoom — это веб-приложение для повышения концентрации и продуктивности. Оно объединяет личное планирование, таймер фокус-сессий и совместную работу в одном сервисе. Проект можно развивать от простого MVP до полноценной платформы для учебы, работы и командной концентрации.
